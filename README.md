# recursive-descent-parser
A recursive-descent parser in C, that evaluates arithmetic expressions

## what does it do?

- [x] addition
- [x] subtraction
- [x] single-digit numbers
- [x] multi-digit numbers
- [x] multiplication
- [x] division
- [x] negation
- [x] parentheses
- [x] exponentiation
- [x] floating point
- [x] error handling
